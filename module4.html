Module 4 - HTML Comments

In this lesson, you will learn how to use HTML comments to add notes and explanations to your code.

Follow the instructions below to complete the exercise:

  1. Open the code editor below and locate the <head> element.
  2. Add a comment inside the <head> element with the text: <!-- Add page title here -->. This comment will remind you to set the title of the page.
  3. Now, find the <body> element.
  4. Add a comment above the <h1> heading with the text: <!-- Main heading goes here -->. This comment will indicate where the main heading of the page should be placed.
  5. Next, add another comment above the <p> paragraph with the text: <!-- Introduction paragraph goes here -->. This comment will guide you to add an introductory paragraph.
  6. Now, locate the <img> tag within the <body> element.
  7. Add a comment above the <img> tag with the text: <!-- Insert image here -->. This comment will prompt you to include an image in the page.
