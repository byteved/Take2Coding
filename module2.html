Module Two - Html Attributes

In this lesson, you will learn about HTML attributes. Attributes provide additional information to HTML elements and are placed inside the opening tag.

Follow the instructions below to complete the code:

  1. Add the doctype declaration: Add the following line at the very beginning of your document: <!DOCTYPE html>
  2. Create an HTML document: Add the <html> element as the root of your document, right after the doctype declaration.
  3. Head to the head: Inside the <html> element, add a <head> element. This is where you put meta-information about the document.
  4. Title your page: Inside the <head> element, add a <title> element with the title "My First Web Page".
  5. Close the head: Close the <head> element using </head>.
  6. Into the body: Create a <body> element inside the <html> element. This is where you put the visible content of your page.
  7. Heading it up: Inside the <body> element, add an <h1> heading with the text "Hello, World!"
  8. Add an image: Add an <img> element with a src attribute and set its value to "example.jpg".
  9. Create a link: Add an <a> element with an href attribute and set its value to "https://www.example.com". Inside the <a> element, write "Visit Example".
  10. Close the body tag: Add a </body> to close the body tag.
  11. Finish the website: Add </html> to close the html tag.
