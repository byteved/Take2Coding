HTML Introduction

In this lesson, you will learn how to create a basic HTML document and display "Hello, World!" on the web page.

Follow the instructions below to complete the code:

  1. Add the doctype declaration<!DOCTYPE html> at the top of your document.
  2. Create an <html> element as the root of the document.
  3. Add a <head> element inside the <html> element.
  4. Inside the <head> element, add a <title> element with the title "My First Web Page". Close it using this: </title>
  5. Close the head element using </head>.
  6. Create a <body> element inside the <html> element.
  7. Inside the <body> element, add an <h1> heading with the text "Hello, World!" Close it using </h1>
  8. Add a <p> paragraph with the text "Welcome to my first web page." Close it with </p>.
  9. Close the <body> element with </body>.
  10. Close the <html> element with </html>.
