HTML Introduction

In this lesson, you will learn how to create a basic HTML document and display "Hello, World!" on the web page.

Follow the instructions below to complete the code:

  1. Add the doctype declaration at the top of your document.
  2. Create an <html> element as the root of the document.
  3. Add a <head> element inside the <html> element.
  4. Inside the <head> element, add a <title> element with the title "My First Web Page".
  5. Create a <body> element inside the <html> element.
  6. Inside the <body> element, add an <h1> heading with the text "Hello, World!"
  7. Add a <p> paragraph with the text "Welcome to my first web page."
